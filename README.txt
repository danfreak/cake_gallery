Description

This Gallery plugin for CakePHP can create albums and upload photos

Instalation

	1. Upload plugin
	2. Add a new album
	3. Upload photos
	4. Enjoy :)
	
Create album and upload any photos, you can access the albums in http://yoursitewithcakephp/gallery, or include it in any location of its pages how often need, just put in the body [Gallery: slug_gallery] that the plugin will automatically substitute by your photo album.

Eg.:

[CakeGallery:my_carnival_brazil_album]
